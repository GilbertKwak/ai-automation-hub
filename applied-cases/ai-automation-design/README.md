# applied-cases/ai-automation-design

## 개요
AI 자동화 설계 및 구현을 위한 심층 프롬프트 모음입니다.  
업무 분석 → AI 적용 설계 → 시스템 아키텍처 → 구현 로드맵까지의 전 과정을 구조화한 템플릿을 포함합니다.

## 파일 구성

| 파일 | 설명 |
|---|---|
| `ai-automation-design-prompt.md` | AI 자동화 설계 및 구현용 심층 프롬프트 (역할 정의 → 7단계 수행 → 출력 형식) |

## 사용 방법

1. `ai-automation-design-prompt.md` 파일을 열어 **입력 정보** 섹션을 채웁니다.
2. 채워진 프롬프트를 LLM (Claude, GPT-4o 등)에 입력합니다.
3. 출력된 7개 섹션을 바탕으로 실제 구현 계획을 수립합니다.

## 적용 대상
- 반복 업무 자동화가 필요한 팀
- AI 도입을 검토 중인 엔지니어 / 컨설턴트
- Notion, GitHub, Python 기반 워크플로우 자동화 프로젝트

## 관련 경로
- `prompts/` — 도메인별 재사용 프롬프트
- `templates/` — 아키텍처 템플릿
