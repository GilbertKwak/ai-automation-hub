## 1. Phase 1 Gate 체크리스트 (PE-1 검증)

### 1-A. 진입 조건 (Entry Criteria)

- [ ] Samsung/SK Hynix LOI (Letter of Intent) 수령 완료
- [ ] 초도 HBM3/HBM3E 샘플 Lot (≥500 units) 확보
- [ ] KGD 테스트 장비 캘리브레이션 완료 (ATE: Advantest T2000 또는 동급)
- [ ] PSR(Post-Salvage Rework) 라인 투자 승인 ($2.4M 예산 확정)
- [ ] 품질 기준서 v1.0 내부 승인

### 1-B. 수율 측정 방법론

```
수율 측정 공식:
  Salvage_Yield(%) = (Grade_A_Units + Grade_B_Units) / Total_Input_Units × 100

등급 분류 기준 (신규 수립 — W1 완료 목표):
  Grade A: 모든 BW 채널 정상, 열 특성 ΔT ≤ 5°C → HBM3E Down-binning
  Grade B: BW 채널 ≥ 75% 정상, ΔT ≤ 8°C   → Micro-bump KGD
  Grade C: BW 채널 < 75% 또는 ΔT > 8°C    → 원자재 재활용 (회수가치 $12/unit)
  Reject:  기계적 손상 또는 치명 오류         → 폐기
```
